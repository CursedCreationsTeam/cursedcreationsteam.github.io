Unlimited Worldgen ("128 bit")

Manifold is necessary for these mods. Install Manifold as a JAR mod via the "Add to Minecraft.jar" option in MultiMC-based launchers. Early versions of Infdev 20100630-1 128-bit also requires BigDecimalMath which is used to do advanced math with BigDecimals, however it's included by default.

Note: These versions are very CPU and RAM-intensive compared to their vanilla counterparts. Do NOT expect to have the same performance as typical 64-bit mods, even at spawn.
